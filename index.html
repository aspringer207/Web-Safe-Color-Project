About

About this collection...

These 216 colors are known as "Web Safe" HTML Colors.

The R, G, and B values that comprise the hex codes each fall into one of six incremental values:

  1. 0, which will always be either 00 or 0, eg. #00FFFF
  2. 51, which is written as 33 in hex code, eg. #330000
  3. 102, which is written as 66 in hex code, eg. #660000
  4. 153, which is written as 99 in hex code, eg. #990000
  5. 204, which is written as CC in hex code, eg. #CC0000
  6. 255, which is written as FF in hex code, eg. #FF0000

Aren't web-safe colors obsolete?

While modern graphics displays solve the most pressing concern addressed by web-safe colors,
there are a few other reasons to use these shades:

Contrast

Since web-safe colors are comprised of regular incremental values, it's relatively simple to choose appropriately contrasting text to ensure readability.
This is especially important for readers with low vision and those who are color-blind.

Variation

There are 16, 581, 375 colors made with hex values.
While 216 colors seems small, especially compared to sixteen million, the evenly spaced intervals provide a relatively wide array of color choices. Plus, incremental values make it easy to find proportional colors like the ones in the table below.

Proportional Colors
Hex Ratio Result
#6633FF 2:1:6
#6633CC 2:1:5
#663399 2:1:4
#663366 2:1:3
#663333 2:1:2
The ratios in the second column reflect the proportions of red, green, and blue, in that order.

Learning

Web-safe hex values are ideal values from which to learn and memorize hex values. Understanding hex values can be tricky at first, but it's a helpful and widely supported tool that can make any designer's life easier.

Convenience

Plus, since the second, fourth, and sixth digit in every web-safe hex code is the same as the digit that precedes it, declaring a variable with a web-safe value is as easy as

  1. /*my stylesheet*/
    .example{color: #6}
    add the first digit of your red value...
  2. /*my stylesheet*/
    .example{color: #63}
    then the green...
  3. /*my stylesheet*/
    .example{color: #639}
    and then the blue!
